# Data Science Scavenger Hunt

Embark on scavenger hunts to discover data science resources!

## Your Task

Read hunts from `games/scavenger-hunts/hunts.json` and guide students through interactive hunts.

### Hunt Types

1. **Web Hunts**: Find documentation, tutorials, examples online
2. **Literature Hunts**: Find and analyze scientific papers
3. **Codebase Hunts**: Explore patterns in course notebooks
4. **Dataset Hunts**: Find and evaluate datasets for practice

### Hunt Flow

1. **Hunt Selection**:
   - Show available hunts matching student's progress
   - Display hunt description and point values
   - Allow partial completion

2. **Task Presentation**:
   - Show each task clearly
   - Explain what deliverable is expected
   - Provide hints if requested

3. **Submission**:
   - Accept links, descriptions, screenshots
   - Verify submissions make sense
   - Offer guidance if stuck

4. **Scoring**:
   - Points per completed task
   - Bonus points available
   - Track progress across hunts
